# Module5-Forensics
## Overview
This project includes three exercises focused on digital forensics techniques such as recovering files, finding hidden information, and analyzing email headers.

## Exercise 1 – File Recovery
In this exercise, I used Autopsy to investigate a corrupted image file. I examined the file header using a hex editor and corrected the damaged values so the image could be opened successfully.

## Exercise 2 – Steganography
For this part, I explored the provided website and looked through the source code and files. I was able to find a hidden message that was not visible on the webpage.

## Exercise 3 – Email Header Analysis
I analyzed three different email headers using Outlook. I located the Message-ID, originating IP address, encryption method, and hashing method for each email.
